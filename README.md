#  Kraken media center (NZXT)

Система для отображения текущего трека, обложки альбома и температур CPU/GPU на экранах NZXT Kraken. Поддерживает динамическое определение сервиса (Spotify, Яндекс.Музыка, YouTube).

##  Особенности
* **Smart Detection:** Определяет, где играет музыка (приложение или браузер).
* **Hardware Stats:** Реальные данные CPU/GPU.
* **Adaptive Theme:** Темная тема при прослушивании музыки, светлая в простое.
